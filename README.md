# Hive
Clean Hive implementation in python. Featuring hot graphics I bought over at  iconfinder.com for $8.

Basic AI using minmax search.

## Running

    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ./run.sh
